
# Lunch de NiceShot  
〜“勝ち飯”が、次のナイスショットを呼ぶ〜

---

## サービス概要  
「Lunch de NiceShot」は、ゴルフラウンドで楽しめるランチ×スコア記録アプリです。  
プレーだけでなく、ハーフ後のランチもゴルフの醍醐味のひとつ。  
スコアは努力の結果。だけど時には、神様にもジンクスにも頼りたくなってしまう。  
そんな時は “あの日の勝ち飯” を食べて、後半戦も笑顔でナイスショット！

---

## このサービスへの思い・作りたい理由  
ゴルフの楽しみは、プレーだけでなくハーフ後のランチ時間にもあります。  
前半のスコアを振り返りながら仲間と食事を囲むあのひととき。  
「ビール飲んじゃったら午後が危ないな…」なんて会話もまた一興です。  

ただ、既存のスコア管理アプリは“競技寄り”で、プレーを支える食事や気分まではフォーカスされていません。  
ならば、ランチもスコアと一緒に残せるようにしたいと思いました。  

スコアとランチを紐づけて記録すれば、  
「この前は冷やし中華の日にベストスコアだった！」など、  
ジンクスや神頼みのような楽しみ方も生まれます。  

いい日も悪い日も、前向きに楽しめるゴルフライフを記録できるアプリを作りたいと思いました。

---

## ユーザー層について  

### ① エンジョイゴルファー層（メインターゲット）  
**特徴：** 仲間と楽しくプレーし、雰囲気を大切にする層。  
**理由：** 既存のスコアアプリは数字や分析中心で、“楽しい記録”には向いていません。  
Lunch de NiceShotは、ランチも含めてゴルフを思い出に残したい人に最適です。  

### ② ビギナーゴルファー層（初心者・女性ゴルファーなど）  
**特徴：** スコアを細かく管理するよりも、プレーの雰囲気や体験を楽しみたい層。  
**理由：** 写真投稿や日記のような感覚で使えるシンプルな操作なので、  
「スコア管理アプリは難しそう…」という初心者でも気軽に記録できます。  
また、SNS文化に馴染んでいる層にも自然に受け入れられます。  

### ③ 中級ゴルファー層（スコアも意識する実力派）  
**特徴：** 練習や分析もするが、時には“気分転換”も重視したい層。  
**理由：** スコアとランチをセットで記録することで、  
「好スコアの日の共通点」＝自分の“勝ち飯パターン”を振り返ることができ、  
メンタルコンディションの維持にも役立ちます。

## サービスの利用イメージ  

### 通常プレーの場合  
1. 前半プレーを終えたら、ハーフスコアを入力。  
　→ 調子を振り返りながら、ランチタイムへ。  
2. ランチ提案 ＆ 記録。  
　→ 食べたランチを写真付きで保存。“勝ち飯”として残せます。  
　→ メニューに迷った時は、ランチルーレットで今日の一品をくじ引き！  
3. 後半スコアを登録し、トータルスコアを確認。    

### スループレーの場合  
1. 前半・後半のスコアを続けて記録。  
2. プレー後にランチ提案 ＆ 記録。  
　→ その日の成果や気分に合わせて、締めのランチを記録できます。  

### 得られる価値  
- スコアとランチの記録を通して、“自分だけのゴルフ日記”が作れる。  
- ランチルーレットで、毎回ちょっとしたワクワクを楽しめる。  
- スルーでも通常でも、ラウンドの流れに自然に寄り添う設計。  
- スコアに関わらず、ゴルフを前向きに楽しめる。

## ユーザーの獲得について  
身近なゴルフ仲間同士の口コミや紹介を想定しています。  
ラウンド中やプレー後に「このアプリおもしろいよ」と見せ合えるような、会話をきっかけに広めていくスタイルです。  

また、SNS共有機能は将来的な拡張機能として検討しています。  

---

## サービスの差別化ポイント・推しポイント  

1. **スコアだけでなく「気分」や「ランチ」も記録できる**  
既存のスコア管理アプリは競技志向が強く、数値データ中心の設計が多いです。  
一方で「Lunch de NiceShot」は、スコア＋ランチ＋その日の気分をセットで記録できます。  
プレーの結果だけでなく、ゴルフの“楽しさそのもの”を残せるのが特徴です。  

2. **「勝ち飯」から自分の好調パターンが見える**  
スコアとランチを紐づけることで、自然と自分の“勝ち飯傾向”が見えてきます。  
「オムライスの日は調子が良い」「カツカレーは攻めすぎた」など、  
ジンクス的な発見がモチベーション維持やメンタル面の支えにもつながります。  

3. **通常プレーもスループレーもカバー**  
ハーフ後のランチタイムがある通常プレーだけでなく、  
スループレーにも対応しており、プレー後にランチ提案＆記録ができます。  
どんなスタイルのラウンドでも“食とスコアの記録”が楽しめます。

## 機能候補  

### MVPリリース  
- ユーザー登録／ログイン  
- スコア記録（前半・後半・トータル）  
- ランチ記録（写真・コメント付き）  
- 勝ち飯ハイライト（良スコア日のランチ表示）  
- 記録一覧・詳細表示  
- ランチルーレット（メニューが決まらない時のくじ）  
- スループレー対応（プレー後のランチ提案＆記録）  
- トータルスコア自動集計、平均スコア表示  

### 本リリース  
- 検索機能（ゴルフ場・メニューなど）  
- グループ登録（同伴者の記録）  
- ジンクス分析（スコア×ランチ傾向）  
- SNS共有機能  
- 管理者画面
